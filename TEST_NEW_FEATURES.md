# 新功能测试清单

## 1. 最大效率点标记配置测试

### 测试项目：
- [ ] 打开页面，确认"标记最大效率点"复选框可用
- [ ] 确认有"最大点形状"下拉选择框，包含：圆形、三角形、矩形、旋转矩形、星形、十字
- [ ] 确认有"最大点大小"数值输入框，范围3-10，默认值6
- [ ] 选择不同形状，观察图表上最大效率点的形状变化
- [ ] 调整大小数值（3-10），观察图表上最大效率点的大小变化
- [ ] 验证最大点颜色与所属曲线颜色一致
- [ ] 验证最大点无边框（或边框颜色与填充色相同）
- [ ] 添加多条曲线，每条曲线的最大点都正确显示
- [ ] 取消"标记最大效率点"复选框，所有最大点标记消失

### 验证标准：
- 最大点形状可配置且即时生效
- 最大点大小可配置且即时生效
- 最大点无明显边框
- 最大点颜色跟随曲线颜色

## 2. 字体大小独立配置测试

### 测试项目：
- [ ] 点击"字体大小设置"折叠面板，展开设置区域
- [ ] 确认有5个字体大小输入框：
  - 图表主标题 (默认18px, 范围8-32)
  - 轴标题 (默认14px, 范围8-24)
  - 轴刻度值 (默认12px, 范围8-20)
  - 图例文字 (默认12px, 范围8-20)
  - 提示文字 (默认12px, 范围8-20)
- [ ] 修改"图表主标题"字体大小，观察主标题文字大小变化
- [ ] 修改"轴标题"字体大小，观察X轴和Y轴标题文字大小变化
- [ ] 修改"轴刻度值"字体大小，观察X轴和Y轴刻度数字大小变化
- [ ] 修改"图例文字"字体大小，观察图例文字大小变化
- [ ] 修改"提示文字"字体大小，鼠标悬停在数据点上，观察提示框文字大小变化
- [ ] 导出配置JSON，验证字体设置已包含
- [ ] 导入配置JSON，验证字体设置正确恢复
- [ ] 刷新页面，验证localStorage保存的字体设置正确恢复

### 验证标准：
- 所有字体大小设置可独立配置
- 修改后即时生效
- 配置可导出/导入
- localStorage自动保存和恢复

## 3. 坐标轴刻度间隔配置测试

### 测试项目：
- [ ] 点击"坐标轴刻度间隔"折叠面板，展开设置区域
- [ ] 确认有"X轴最小刻度间隔"输入框（占位符：自动计算）
- [ ] 确认有"Y轴最小刻度间隔"输入框（占位符：自动计算）
- [ ] 输入X轴刻度间隔（如：10），观察X轴刻度按此间隔显示
- [ ] 输入Y轴刻度间隔（如：5），观察Y轴刻度按此间隔显示
- [ ] 清空刻度间隔输入框，观察恢复为自动计算
- [ ] 导出配置JSON，验证刻度间隔设置已包含
- [ ] 导入配置JSON，验证刻度间隔设置正确恢复

### 验证标准：
- X/Y轴刻度间隔可独立配置
- 输入后即时生效
- 清空时自动计算
- 配置可导出/导入

## 4. 曲线数据一键折叠测试

### 测试项目：
- [ ] 确认"曲线数据管理"区域有"📁 折叠全部"按钮
- [ ] 点击"📁 折叠全部"按钮，所有曲线面板内的折叠区域（批量粘贴、逐点添加、数据表）都折叠
- [ ] 按钮文字变为"📂 展开全部"
- [ ] 点击"📂 展开全部"按钮，所有曲线面板内的折叠区域都展开
- [ ] 按钮文字变回"📁 折叠全部"
- [ ] 手动展开/折叠单个面板，再点击一键按钮，验证行为合理
- [ ] 添加多条曲线，验证一键折叠/展开对所有曲线生效

### 验证标准：
- "折叠全部/展开全部"按钮可用
- 按钮文字随状态切换
- 一键操作影响所有曲线的所有折叠面板
- 不影响数据本身
