# 最大效率点标记与表格高亮优化 - 验收清单

## 1. 图例显示 ✓
- [ ] 打开 index.html
- [ ] 添加一条曲线并输入数据
- [ ] 确认图例中**不显示**"最大效率点"独立条目
- [ ] 只显示曲线名称，不显示"XXX - 最大效率"

## 2. 图上标记 ✓
- [ ] 确认最大效率点标记颜色与曲线颜色一致（核心部分）
- [ ] 标记有对比色边框（黑色或白色）增强可见性
- [ ] 标记较大，容易识别（圆形，半径约9px）
- [ ] 悬停时标记变大

## 3. 标记开关 ✓
- [ ] 全局设置中有"标记最大效率点"开关（不是"高亮"）
- [ ] 默认状态为开启
- [ ] 关闭开关后，图上不显示最大效率点标记
- [ ] 开启开关后，图上显示最大效率点标记

## 4. 数据表高亮 ✓
- [ ] 在曲线数据表中，最大效率点所在行有明显高亮
- [ ] 高亮样式：金色背景、加粗、左侧金色边框、星星图标
- [ ] 深色模式下高亮样式适配良好

## 5. 数据变化响应 ✓
- [ ] 添加数据点后，高亮自动更新到新的最大效率点
- [ ] 删除数据点后，高亮自动更新
- [ ] 批量粘贴数据后，高亮正确显示
- [ ] 清空数据后，高亮消失

## 6. 并列最大值处理 ✓
- [ ] 添加多个效率相同的点
- [ ] 确认选择功率（X值）最小的点进行标记和高亮

## 7. 曲线显隐 ✓
- [ ] 隐藏曲线后，图上标记不显示
- [ ] 隐藏曲线后，数据表中高亮**仍然显示**
- [ ] 显示曲线后，图上标记重新显示

## 8. 曲线颜色变更 ✓
- [ ] 修改曲线颜色
- [ ] 确认最大效率点标记核心颜色跟随曲线颜色变化
